# Entrega lista - página de portafolio

## Archivos principales
- `portafolio.html`
- `assets/css/portafolio.css`
- `assets/img/*`

## Cómo integrarlo
1. Copia `portafolio.html` en la misma carpeta donde está tu `index.html`.
2. Copia la carpeta `assets` para mantener imágenes y estilos.
3. Verifica que en tu CV exista el enlace:
   `href="portafolio.html"`
4. Sube todo al repositorio y publica con GitHub Pages.

## Qué cubre esta página
- Página de portafolio responsive.
- Imágenes de trabajos anteriores.
- Diseño profesional con Bootstrap + CSS + Flexbox.
- Estructura lista para evaluación visual.

## Qué debes completar tú para el puntaje total
- Publicar el repositorio en GitHub Pages con el CV como página principal.
- Hacer los forks y commits solicitados por la prueba.
